## Tech Stack

- React JS with Typescript
- Tailwind Styled Component
- Redux Toolkit
- React Router

### Features

- This application utilizes [Makeup API](https://makeup-api.herokuapp.com/).

## How to run locally

clone the repo,
`npm install` then `npm start`,
then listen to `localhost:3000`
